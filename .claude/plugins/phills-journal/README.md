# Phill's Journal

Skill for writing and publishing editions of **Phill's Journal: Insights & Innovations**, the Ask Phill ecommerce newsletter.

## Setup

1. Set `NOTION_API_KEY` in your environment
2. Update the database ID in `skills/phills-journal/references/notion-api.md`
3. Install the `journal-cover` plugin for cover image generation

## Usage

Ask Claude to write a new journal edition:

- "Write a new Phill's Journal edition"
- "Create a journal edition about Shopify's latest updates"
- "Draft a Phill's Journal covering Flow improvements and Klaviyo tricks"
